# Sakura Агент — Обновление (Фазы 0-2)

## Что изменилось

- `config.py` — все секреты из `.env`, нет захардкоженного IP
- `core/agent.py` — токен в каждом WS-сообщении + обработка `mood_update`
- `ui/overlay.py` — SphereCore с mood-вектором: цвет, погода, пульс, лепестки
- `ui/app.py` — новый сигнал `moodUpdate` в шине
- `sakura.py` — подключён `moodUpdate → overlay.set_mood()`

## Быстрый старт

```bash
cp .env.example .env
nano .env  # заполни VPS_WS_URL и WS_TOKEN
```

`.env`:
```
VPS_WS_URL=ws://144.31.47.139:8765    # или wss:// после настройки TLS
DEVICE_ID=laptop                       # или pc
WS_TOKEN=<тот же токен что WS_SECRET на VPS>
WHISPER_DEVICE=cuda
```

## Что делает орб теперь

| Состояние орба | Когда |
|---|---|
| Тёплый розово-золотой | Позитивное настроение (valence > 0) |
| Холодный синий | Тревога, раздражение (valence < 0) |
| Буря внутри сферы | Взволнована и негативна |
| Туман | Грустно, одиноко |
| Золотые искры (shimmer) | Спокойна и довольна |
| Быстрые лепестки | Высокое возбуждение, радость |
| Медленный пульс | Ночной режим (22:00–07:00) |
| Микропауза ритма | Сомнение, неопределённость |
